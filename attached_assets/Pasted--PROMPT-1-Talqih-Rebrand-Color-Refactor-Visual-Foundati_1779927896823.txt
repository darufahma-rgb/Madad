## Filosofi Visual Talqih (BACA DULU SEBELUM KODING)

### Mood
*Calm, intelligent, premium, focused, subtly spiritual, companion-like.*
**Bukan**: hustle, productivity app, sales-y, ramai.

### Color Discipline (NON-NEGOTIABLE)

Saat ini codebase punya **warna tersier yang nyelinap** dari early development:
- Rose/red (`bg-rose-600`, `text-rose-600`, `text-red-400`) — untuk error & destructive action
- Mint green (`mint-500`, `mint-600`) — untuk success
- Beberapa accent warna lain

**Aturan baru:**
- Background gradient pakai HANYA: purple (violet-600, violet-700, violet-900), gold (sangat subtle, max 6% opacity), dan night base
- Action button: violet only
- Destructive/error: pakai **purple-rose muted** (#C97B9B) — bukan merah pure. Lebih elegan, sesuai sanctuary mood.
- Success: pakai **gold sutera** (#D4B27D) — bukan green. Sesuai filosofi spiritual.
- Hapus semua referensi ke red, rose pure, mint, emerald, sky, amber, yellow

### Font Hierarchy untuk Arab

Saat ini cuma pakai 1 font Arab (Noto Naskh) untuk semua. Sekarang dibagi jadi 3 tier:

| Tier | Pakai untuk | Font |
|---|---|---|
| **Display Klasikal** | Atsar, niat, ayat emosional, salam besar | **Aref Ruqaa** |
| **Display Modern** | Bismillah ornament, judul section Arab, tagline brand | **Reem Kufi** |
| **Body (existing)** | Paragraf dalil, terjemah, body kitab | **Noto Naskh Arabic** (tetap) |
| **UI Inline** | Chip madzhab, tag, label kecil, footnote | **Noto Kufi Arabic** |

### Pattern di Hero

Tambah subtle Islamic geometric pattern sebagai SVG overlay di hero section dengan opacity **3-5%**. Cuma kerasa sebagai *texture*, gak boleh kelihatan jelas. Pilihan pattern: 8-point star (Khatam Sulaymani) atau zellige tile. Gue rekomendasi 8-point star karena lebih clean dan menggambarkan kesempurnaan.

## Catatan Penting

### Backward Compatibility
User yang sudah pakai MADAD versi sebelumnya tidak boleh kehilangan data mereka. localStorage keys tetap pakai prefix `madad_*`. Kalau lo tergoda untuk "merapikan" jadi `talqih_*` — **JANGAN**. Itu akan reset semua data user.

### Visual Test Manual
Setelah selesai, ambil screenshot landing page hero — sebelum dan sesudah. Compare:
- **Sebelum:** dark purple solid dengan blob lampu warm
- **Sesudah:** dark purple sedikit lebih warm, dengan texture 8-point star subtle, dan blob masih ada

Kalau hasilnya hero terasa lebih "berisi tapi tetap tenang" — berhasil. Kalau hero kerasa lebih ramai/distracting — pattern opacity terlalu tinggi, turunkan ke 3%.

### Yang Akan Dilakukan di Prompt Selanjutnya

- **Prompt 2**: Landing Page Baru (5 section pendek, sample Maddah Nahwu accessible publik)
