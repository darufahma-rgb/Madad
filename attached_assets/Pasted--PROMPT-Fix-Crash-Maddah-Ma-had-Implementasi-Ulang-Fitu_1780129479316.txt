# PROMPT: Fix Crash Maddah Ma'had + Implementasi Ulang Fitur Ma'had

## Konteks Masalah

Ada crash saat buka detail maddah Ma'had karena:
1. Ada sisa kode/data dari prompt Ma'had sebelumnya yang formatnya tidak kompatibel
2. File `mahad-data.jsx` dan `mahad-maddah.jsx` belum ada
3. Level Ma'had belum ada di `data.jsx`

Prompt ini: BERSIHKAN sisa lama, lalu implementasi ulang dari awal dengan benar.

---

## LANGKAH 0 — BERSIHKAN SISA LAMA

### 0.1 — Hapus maddah Ma'had dari `src/maddah-data.jsx`

Buka `src/maddah-data.jsx`. Cari dan hapus semua maddah yang punya:
- `jenjang: ["idad"]` atau `jenjang: ["tsanawi"]`
- `category: "umum"` dengan id seperti "matematika-idad", "fisika-tsanawi", dll
- Apapun yang berhubungan dengan Ma'had yang ditambahkan sebelumnya

Pastikan `MADDAHS` array hanya berisi 35-36 maddah S1 Al-Azhar yang original.

### 0.2 — Hapus referensi Ma'had dari `src/data.jsx`

Cari dan hapus level-level ini kalau sudah ada:
```
"idad_1", "idad_2", "idad_3"
"tsanawi_1_adabi", "tsanawi_1_ilmi"
"tsanawi_2_adabi", "tsanawi_2_ilmi"  
"tsanawi_3_adabi", "tsanawi_3_ilmi"
```

### 0.3 — Hapus file lama kalau ada

Kalau ada file `src/mahad-data.jsx` atau `src/pages/mahad-maddah.jsx` dari sebelumnya — hapus dulu, kita buat ulang yang benar.

### 0.4 — Hapus script tags Ma'had dari `index.html`

Kalau ada baris seperti:
```html
<script ... src="src/mahad-data.jsx"></script>
<script ... src="src/pages/mahad-maddah.jsx"></script>
```
Hapus dulu. Kita tambahkan lagi nanti di langkah terakhir.

### 0.5 — Test dulu

Setelah bersih, refresh website — halaman maddah S1 harus normal kembali (tidak ada error boundary).

```js
{ id: "tsanawi_2_ilmi",  label: "Tsanawi Kelas 2 — Ilmi",  short: "Tsanawi 2 Ilmi",  arabic: "ثانوي ٢ علمي" },
{ id: "tsanawi_3_adabi", label: "Tsanawi Kelas 3 — Adabi", short: "Tsanawi 3 Adabi", arabic: "ثانوي ٣ أدبي" },
{ id: "tsanawi_3_ilmi",  label: "Tsanawi Kelas 3 — Ilmi",  short: "Tsanawi 3 Ilmi",  arabic: "ثانوي ٣ علمي" },
```

Tambahkan helper functions setelah LEVELS:

```js
const MAHAD_LEVEL_IDS = [
  "idad_1","idad_2","idad_3",
  "tsanawi_1_adabi","tsanawi_1_ilmi",
  "tsanawi_2_adabi","tsanawi_2_ilmi",
  "tsanawi_3_adabi","tsanawi_3_ilmi",
];
const isMahadLevel   = (level) => MAHAD_LEVEL_IDS.includes(level);
const isIdadLevel    = (level) => level?.startsWith("idad");
const isTsanawiLevel = (level) => level?.startsWith("tsanawi");
const isIlmiLevel    = (level) => level?.includes("ilmi");
const isAdabiLevel   = (level) => level?.includes("adabi");

Object.assign(window, {
  MAHAD_LEVEL_IDS, isMahadLevel, isIdadLevel,
  isTsanawiLevel, isIlmiLevel, isAdabiLevel,
});
```
