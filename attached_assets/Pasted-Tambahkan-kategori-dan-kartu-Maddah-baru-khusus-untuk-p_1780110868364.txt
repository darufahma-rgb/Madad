## A. TAMBAH FILTER/TAB BARU

Di filter kategori halaman Maddah, tambahkan
tab baru paling kiri: "Ma'had"

Sub-filter di bawah tab Ma'had:
→ Semua
→ Maddah Agama
→ Maddah Umum — Ilmi
→ Maddah Umum — Adabi

## C. FILTER OTOMATIS BERDASARKAN PROFIL

Ketika user login dengan profil Ma'had, halaman
Maddah otomatis menampilkan filter "Ma'had" aktif
dan hanya tampilkan Maddah yang relevan:

- User I'dadi → tampilkan Maddah berlabel "I'dadi"
- User Tsanawi Ilmi → tampilkan Maddah Agama +
  Kartu 7, 8, 9, 10, 11
- User Tsanawi Adabi → tampilkan Maddah Agama +
  Kartu 7, 8, 12, 13, 14, 15, 16

User tetap bisa lihat semua Maddah kalau klik
"Lihat Semua" — filter hanya default, bukan lock.

Jangan ubah apapun yang sudah ada untuk Masisir.
