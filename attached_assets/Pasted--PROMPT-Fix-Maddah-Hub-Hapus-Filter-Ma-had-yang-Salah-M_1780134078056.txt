# PROMPT: Fix Maddah Hub — Hapus Filter Ma'had yang Salah

## Masalah

Di halaman `/maddah` (Maddah Hub), Replit menambahkan filter "Ma'had" dan
checkbox "Tampilkan hanya untuk Tsanawi" — tapi filter ini tidak berfungsi
karena `MaddahHubPage` hanya membaca dari `MADDAHS` (S1), bukan `MAHAD_MADDAH`.

Hasilnya: user klik filter "Ma'had" → "Tidak ada Maddah yang cocok dengan filter."

## Solusi

Maddah Ma'had punya halaman sendiri di `/mahad-maddah`.
Jadi di `MaddahHubPage`:
1. Hapus filter "Ma'had" dari category tabs
2. Hapus checkbox "Tampilkan hanya untuk Tsanawi"
3. Untuk user Ma'had, tambahkan banner redirect ke `/mahad-maddah`

---

## STEP 1 — Hapus filter Ma'had dari `src/pages/maddah-hub.jsx`

### 1.1 — Hapus state yang tidak perlu

Cari dan hapus state-state ini kalau ada:
```js
// HAPUS kalau ada:
const [showOnlyTsanawi, setShowOnlyTsanawi] = useState(false);
const [mahadFilter, setMahadFilter] = useState(false);
```

### 1.2 — Hapus dari CATEGORY_TABS / filter chips

Cari bagian di mana category tabs di-render (biasanya ada array seperti
`[{id:"all",...}, {id:"qurani",...}, ...]`).

Hapus entry untuk "mahad" atau "Ma'had":
```js
// HAPUS entry ini kalau ada:
{ id: "mahad", label: "Ma'had", icon: "🕌" },
// atau apapun yang id-nya "mahad"
```

### 1.3 — Hapus checkbox "Tampilkan hanya untuk Tsanawi"

Cari dan hapus seluruh elemen checkbox yang berkaitan dengan Tsanawi/Ma'had:
```jsx
// HAPUS ini (atau yang serupa):
<input type="checkbox" ... /> Tampilkan hanya untuk Tsanawi
```

### 1.4 — Pastikan filter logic bersih

Di fungsi filter `MADDAHS.filter(m => {...})`, pastikan tidak ada kondisi
yang reference `mahad`, `tsanawi`, atau `idad`:

```js
// Filter yang bersih — hanya S1:
const filtered = MADDAHS.filter(m => {
  if (activeCategory !== "all" && m.category !== activeCategory) return false;
  if (showOnlyMyFaculty && profile?.faculty && profile.faculty !== "umum") {
    if (!m.fakultas.includes(profile.faculty)) return false;
  }
  if (searchQuery.trim()) {
    const q = searchQuery.toLowerCase();
    if (!m.name.toLowerCase().includes(q)
      && !m.nameArabic.includes(searchQuery)
      && !m.description.toLowerCase().includes(q)) return false;
  }
  return true;
});
```
