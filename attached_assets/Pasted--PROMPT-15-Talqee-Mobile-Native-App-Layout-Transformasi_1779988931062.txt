# PROMPT 15 — Talqee: Mobile Native-App Layout

> Transformasi tampilan mobile (<768px) supaya terasa seperti native app, bukan website desktop dikecilin.
> Desktop (md: ke atas) TIDAK boleh berubah sama sekali — semua perubahan wajib di dalam breakpoint mobile.
> Kerjakan per layer, dari foundation ke komponen.

---

## Prinsip Native Mobile (Pegang Ini)

```
WEBSITE MOBILE (jelek)     →    NATIVE APP MOBILE (target)
─────────────────────────       ──────────────────────────
Padding besar (64px)            Padding compact (16-24px)
Card padding p-6/p-7            Card padding p-3/p-4
Heading font-display besar      Heading proporsional
1 layout untuk semua screen     Layout khusus mobile
Hamburger menu                  Bottom tab bar
Modal di tengah                 Bottom sheet dari bawah
Scroll panjang tiap section     Konten padat, density tinggi
Tap target kecil                Tap target min 44px
```

**Aturan wajib:**
- Semua perubahan di dalam `@media (max-width: 767px)` atau class `md:` prefix
- Desktop TIDAK berubah — test setelah selesai di desktop
- Touch target minimum 44px height untuk semua tombol interaktif
- Font body minimum 14px (cegah iOS auto-zoom saat tap input)
- Input font-size minimum 16px (cegah iOS zoom)

## LAYER 2 — Bottom Tab Bar di `src/layout.jsx`

Navigasi utama mobile pindah ke bawah. Hapus hamburger menu lama untuk member.

### STEP 2.1 — Komponen `MobileTabBar`

Tambah setelah closing tag `Navbar` di layout.jsx:

```jsx
const MobileTabBar = ({ session, profile, path }) => {
  if (!session || !profile?.onboarded) return null;

  const tabs = [
    { to: "/dashboard",       icon: "home",     label: "Beranda"   },
    { to: "/maddah",          icon: "layers",   label: "Maddah"    },
    { to: "/paths/muqaranah", icon: "scale",    label: "Muqaranah" },
    { to: "/kurasah",         icon: "notebook", label: "Kurasah"   },
    { to: "/tools",           icon: "sparkles", label: "Tools"     },
  ];

  const active = (to) => {
    if (to === "/dashboard") return path === "/dashboard" || path === "/" || path === "";
    return path.startsWith(to);
  };

  return (
    <nav className="md:hidden fixed bottom-0 left-0 right-0 z-50 border-t border-line"
      style={{
        background: "rgba(10,5,20,0.92)",
        backdropFilter: "blur(20px)",
        WebkitBackdropFilter: "blur(20px)",
        paddingBottom: "var(--safe-bottom)",
      }}>
      <div className="flex items-stretch" style={{ height: "var(--tab-bar-h)" }}>
        {tabs.map(tab => {
          const isActive = active(tab.to);
          return (
            <button key={tab.to} onClick={() => navigate(tab.to)}
              className="flex-1 flex flex-col items-center justify-center gap-0.5 relative"
              style={{ minHeight: 44 }}>
              {/* Active indicator */}
              {isActive && (
                <span className="absolute top-0 left-1/2 -translate-x-1/2 w-8 h-0.5 rounded-full bg-violet-400"/>
              )}
              <Icon name={tab.icon}
                className={`w-[22px] h-[22px] transition-colors ${isActive ? "text-violet-300" : "text-ink-soft"}`}/>
              <span className={`text-[10px] transition-colors leading-none ${isActive ? "text-violet-200 font-medium" : "text-ink-soft"}`}>
                {tab.label}
              </span>
            </button>
          );
        })}
      </div>
    </nav>
  );
};
```
