# PROMPT 6 — Talqih: Misi Interaktif + Starter Pack + Logo AI Rapih

> Ini **Prompt 6** dalam roadmap Talqih (tambahan setelah Prompt 1-5).
> Berisi 3 fitur terpisah yang bisa dikerjakan berurutan.

---

## Konteks

3 perbaikan untuk Talqih (panduan belajar Al-Azhar dengan AI):

1. **Misi/Modul interaktif** — saat user klik "Mulai modul" di Learning Path, dia diarahkan mengerjakan misi sesuai konteksnya (Materi/Praktik/Tour punya flow beda), bukan cuma toggle "Selesai".
2. **Starter Pack** — template perkenalan diri user yang di-copy ke AI di awal sesi, supaya AI langsung "kenal" siapa user (tingkat, fakultas, gaya belajar).
3. **Logo AI rapih** — perbaiki tampilan logo model AI yang sekarang dipaksa jadi putih flat (jelek).

---

# BAGIAN 1 — MISI/MODUL INTERAKTIF

## Masalah Sekarang

Di Learning Path, modul cuma punya tombol "Mulai modul" / "Selesai" yang toggle status. Tidak ada konten/aktivitas nyata. User klik "Selesai" tanpa benar-benar mengerjakan apa-apa.

## Yang Diinginkan

Setiap modul punya **3 jenis (`kind`)** dengan flow berbeda saat dibuka:
- **Materi** → tampilkan konten bacaan + poin kunci, lalu tombol "Tandai sudah dibaca"
- **Praktik** → kasih instruksi langkah + prompt untuk dicoba ke AI, lalu checklist "sudah dicoba"
- **Tour** → walkthrough beberapa langkah (multi-step) mengenalkan tools/fitur

## Implementasi

### STEP 1.3 — Komponen `<ModuleRunner>`

```jsx
const ModuleRunner = ({ pathId, module, onClose, onComplete }) => {
  const content = module.content;
  
  return (
    <div className="fixed inset-0 z-[60] flex items-center justify-center p-4 bg-night-950/80 backdrop-blur-sm" onClick={onClose}>
      <div className="card-glass-strong max-w-2xl w-full max-h-[85vh] overflow-y-auto p-6 md:p-8 relative" onClick={e => e.stopPropagation()}>
        {/* Close */}
        <button onClick={onClose} className="absolute top-4 right-4 text-ink-soft hover:text-ink w-8 h-8 flex items-center justify-center rounded-lg hover:bg-white/5">✕</button>
        
        {/* Header */}
        <div className="mb-6">
          <div className="text-[11px] uppercase tracking-wider text-gold-400 mb-1">{module.kind} · {module.duration}</div>
          <h2 className="font-display text-2xl md:text-3xl font-semibold text-ink">{module.title}</h2>
        </div>

        {/* Content by type */}
        {content?.type === "materi" && <MateriContent content={content}/>}
        {content?.type === "praktik" && <PraktikContent content={content}/>}
        {content?.type === "tour" && <TourContent content={content}/>}
        {!content && <p className="text-ink-muted">Konten modul ini sedang disiapkan.</p>}

        {/* Footer action */}

// ── Tour ──

        <p className="text-sm text-ink-muted leading-relaxed">{stop.body}</p>

        <div className="flex gap-1.5">

          ))}
        </div>
        <button onClick={() => setStep(Math.min(stops.length-1, step+1))} disabled={step===stops.length-1} className="btn-ghost text-sm px-4 py-2 disabled:opacity-30">Selanjutnya →</button>
      </div>
    </div>
  );
};
```

# BAGIAN 2 — STARTER PACK (Perkenalan User ke AI)

## Konsep

Template berisi perkenalan diri user (tingkat, fakultas, jurusan, gaya belajar) yang di-copy ke AI **di awal sesi belajar**. Setelah AI tahu konteks user, semua jawaban berikutnya jadi lebih relevan. Ini "pembuka percakapan" yang bikin AI langsung kenal siapa user.

## Implementasi

### STEP 2.1 — Buat Fungsi Generator di `src/adaptive-prompt.jsx`

Tambahkan fungsi yang generate starter pack dari profile:

```js
const generateStarterPack = (profile, session) => {
  const tingkatan = profile?.level ? (TINGKATAN_LABEL[profile.level] || "thalib") : "thalib";
  const fakultas = profile?.faculty ? (FAKULTAS_LABEL[profile.faculty] || "Al-Azhar") : "Al-Azhar";
  const jurusan = profile?.major ? (JURUSAN_LABEL[profile.major] || "").replace(/^, /, "") : "";
  const nama = session?.name || "";
  
  // Gaya belajar
  let gaya = "kombinasi";
  if (profile?.learningStyle?.length > 0) {
    const labels = profile.learningStyle.map(s => GAYA_BELAJAR_LABEL[s]).filter(Boolean);
    gaya = labels.length > 1 ? labels.slice(0,-1).join(", ") + " dan " + labels[labels.length-1] : (labels[0] || "kombinasi");
  }
  
  // Level bahasa instruction
  const levelBahasa = profile?.level ? (LEVEL_BAHASA_INSTRUCTION[profile.level] || LEVEL_BAHASA_INSTRUCTION["2"]) : LEVEL_BAHASA_INSTRUCTION["2"];

  return `Assalamu'alaikum. Mulai sekarang, kamu akan membantuku belajar materi Al-Azhar. Kenali dulu siapa aku:

📿 PROFIL
- Aku ${nama ? nama + ", " : ""}seorang ${tingkatan}
- Belajar di ${fakultas}${jurusan ? ", " + jurusan : ""}
- Cara belajarku: aku ${gaya}

📖 CARA KAMU MEMBANTUKU
- Bahasa pengantar: Indonesia akademik
- ${levelBahasa}
- Saat menyebut ayat, hadits, atau kaidah: WAJIB sertakan teks Arab asli dengan harakat, lalu terjemahnya
- Saat menyebut istilah teknis: tulis Arab + transliterasi (contoh: قياس (qiyas))
- Saat menyebut ulama: pakai nama transliterasi natural (Imam Syafi'i, Ibnu Hajar)
- Saat menyebut kitab: pakai nama Arab (Al-Umm, Fathul Bari)

⚠️ ADAB PENTING
- Kamu adalah alat bantu, BUKAN pengganti guru/syaikh-ku
- Kalau kamu tidak yakin atau informasinya lemah, katakan dengan jujur
- Jangan men-tarjih (memilih pendapat terkuat) dalam masalah khilafiyah — sajikan semua pendapat, biar aku yang merenungkan bersama guruku

Kalau kamu paham, jawab singkat: "Wa'alaikumussalam, siap membantu." Lalu tunggu pertanyaan pertamaku.`;
};

// Export
window.generateStarterPack = generateStarterPack;
```

# BAGIAN 3 — LOGO AI RAPIH

## Masalah Sekarang

Di `src/ui.jsx` `ToolIcon`, logo di-render dengan `filter: "brightness(0) invert(1)"` — ini **memaksa SEMUA logo jadi putih solid flat**, menghilangkan warna asli & detail logo. Hasilnya jelek/tidak rapih. Plus path logo lokal (`/assets/logo-*.png`) mungkin tidak ada filenya.

## Fix

### STEP 3.1 — Update Logo URL di `src/data.jsx`

Ganti field `logo` tiap AI tool dengan logo resmi dari CDN yang reliable. Gunakan logo dari sumber publik (Wikimedia/official) — atau pakai monogram berwarna sebagai fallback yang lebih rapih.

**Rekomendasi:** karena dependensi ke URL eksternal bisa rapuh, **pakai pendekatan brand-colored monogram** yang lebih kontrol & konsisten. Update `AI_TOOLS` dengan warna brand yang akurat:

```js
// Update tiap tool dengan brandColor & hapus dependency ke logo file lokal yang bermasalah
{ id: "chatgpt",    name: "ChatGPT",    by: "OpenAI",      monogram: "GPT", brandColor: "#10A37F", logo: null, ... },
{ id: "claude",     name: "Claude",     by: "Anthropic",   monogram: "Cl",  brandColor: "#D97757", logo: null, ... },
{ id: "gemini",     name: "Gemini",     by: "Google",      monogram: "G",   brandColor: "#4285F4", logo: null, ... },
{ id: "notebooklm", name: "NotebookLM", by: "Google",      monogram: "NB",  brandColor: "#9168F0", logo: null, ... },
{ id: "perplexity", name: "Perplexity", by: "Perplexity",  monogram: "Px",  brandColor: "#20B8CD", logo: null, ... },
{ id: "deepseek",   name: "DeepSeek",   by: "DeepSeek",    monogram: "Ds",  brandColor: "#4D6BFE", logo: null, ... },
```

(Sesuaikan dengan tools yang ada. brandColor adalah warna resmi tiap brand.)

### STEP 3.2 — Perbaiki Komponen `ToolIcon` di `src/ui.jsx`

Ganti komponen jadi monogram berwarna brand yang clean & konsisten (tidak lagi paksa putih flat):

```jsx
const ToolIcon = ({ tool, size = "w-11 h-11", rounded = "rounded-xl", className = "" }) => {
  const bg = tool.brandColor || "#7C4DFF";
  return (
    <span
      className={`${size} ${rounded} flex items-center justify-center overflow-hidden flex-shrink-0 ${className}`}
      style={{
        background: `linear-gradient(135deg, ${bg}, ${bg}CC)`,
        boxShadow: `0 2px 12px ${bg}33, inset 0 1px 0 rgba(255,255,255,0.15)`,
      }}
    >
      {tool.logo
        ? <img src={tool.logo} alt={tool.name} className="w-full h-full object-contain p-1.5"/>
        : <span className="text-sm font-bold text-white tracking-tight" style={{ textShadow: "0 1px 2px rgba(0,0,0,0.25)" }}>{tool.monogram}</span>
      }
    </span>
  );
};
```

Ini menghasilkan icon **kotak rounded dengan gradient warna brand asli + monogram putih bersih** — rapih, konsisten, dan tetap dalam aesthetic Talqih. Tidak ada lagi logo putih flat yang jelek.

**Catatan:** kalau lo punya file logo PNG asli yang bagus (transparent, berwarna), bisa set `logo: "/assets/logo-xxx.png"` dan `ToolIcon` akan render itu tanpa filter putih. Tapi kalau tidak yakin, biarkan `logo: null` dan pakai monogram berwarna — lebih aman & tetap rapih.

### STEP 3.3 — Verifikasi Tampilan di Semua Tempat

ToolIcon dipakai di banyak tempat (dashboard AI rec, maddah detail, sample nahwu, tour module). Pastikan semua render rapih dengan style baru. Cek tidak ada lagi referensi `filter: brightness(0) invert(1)` yang tersisa.
