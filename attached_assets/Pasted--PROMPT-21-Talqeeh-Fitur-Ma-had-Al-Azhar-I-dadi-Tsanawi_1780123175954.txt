### STEP 1.2 — Helper `isMahad` di `src/data.jsx`

Tambahkan helper function setelah definisi LEVELS:

```js
const MAHAD_LEVEL_IDS = [
  "idad_1","idad_2","idad_3",
  "tsanawi_1_adabi","tsanawi_1_ilmi",
  "tsanawi_2_adabi","tsanawi_2_ilmi",
  "tsanawi_3_adabi","tsanawi_3_ilmi",
];
const isMahadLevel = (level) => MAHAD_LEVEL_IDS.includes(level);
const isIdadLevel  = (level) => level?.startsWith("idad");
const isTsanawiLevel = (level) => level?.startsWith("tsanawi");
const isIlmiLevel  = (level) => level?.includes("ilmi");
const isAdabiLevel = (level) => level?.includes("adabi");

// Export ke window
Object.assign(window, { isMahadLevel, isIdadLevel, isTsanawiLevel, isIlmiLevel, isAdabiLevel, MAHAD_LEVEL_IDS });
```

const getMahadMaddahByJenjang = (level) => {
  if (!level) return [];
  const isIdad = level.startsWith("idad");
  const isTsanawi = level.startsWith("tsanawi");
  const isIlmi = level.includes("ilmi");
  const isAdabi = level.includes("adabi");

  return MAHAD_MADDAH.filter(m => {
    const jenjangMatch = (isIdad && m.jenjang.includes("idad"))
      || (isTsanawi && m.jenjang.includes("tsanawi"));
    if (!jenjangMatch) return false;
    if (!m.jurusan) return true;
    if (isIlmi  && m.jurusan.includes("ilmi"))  return true;
    if (isAdabi && m.jurusan.includes("adabi")) return true;
    if (isIdad) return true;
    return false;
  });
};

const getMahadMaddahById = (id) => MAHAD_MADDAH.find(m => m.id === id) || null;

## Catatan Penting

### Tone untuk Remaja (WAJIB dijaga)
Semua prompt Ma'had dirancang dengan tone yang encouraging dan age-appropriate untuk remaja 12-18 tahun. Jangan ubah frasa seperti "Buat aku benar-benar paham!", "semenarik mungkin", dan "Aku masih remaja" — ini disengaja untuk membuat AI merespons dengan cara yang tepat untuk usia mereka.

### Maddah Umum dalam Bahasa Arab
Tantangan unik Ma'had: matematika, fisika, kimia dll diajarkan DALAM BAHASA ARAB. Semua prompt umum sudah menyertakan instruksi untuk AI memberikan tabel istilah Arab ↔ Indonesia — ini yang paling membantu pelajar Ma'had.

### 17 Maddah Total
- 9 maddah agama (berlaku I'dadi + Tsanawi)
- 2 maddah umum I'dadi (Matematika, Sains)
- 1 maddah umum semua jenjang (Bahasa Inggris)
- 4 maddah Tsanawi Ilmi (Matematika lanjut, Fisika, Kimia, Biologi)
- 5 maddah Tsanawi Adabi (Sejarah, Geografi, Filsafat & Mantiq, Psikologi, Statistik)
